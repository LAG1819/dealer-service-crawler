# dealer-service-crawler
A Python-based web crawler and dashboard system for harvesting and visualizing automotive service data from dealership websites.

This project was developed and designed solely by Luisa-Sophie Gloger within the subject "Web- and Social media Analytics" at Hochschule der Medien - University of Applied Sciences Stuttgart.

This project integrates classic processes of Natural Language Processing and dynamic web crawling.

## Project Overview

### Research Question:
What types of services are offered online by car dealerships?

### Goal:
To dynamically categorize and identify the range of services provided by car dealerships and workshops in Germany, selectable by the user.

### Challenges:

- Highly diverse website structures and layouts
- Crawler and bot protection mechanisms
- Detecting and handling inactive or non-existent websites
